About Us

This project is made in partial fulfillment of the course IT324 scheduled in 4:00PM-5:00PM TF under our instructor, Ma'am. Ma. Concepcion Clemente. This project is being passed every end of the terms, so it is expected for the website to be updated regularly.

This is a website containing all our lecture notes in Web Systems and Technologies. The website has five sections which are World Wide Web, Hypertext Transfer Protocol, Hypertext Markup Language, Cascading Stylesheets, and Javascript. In each sections, there are subtopics which were discussed by our instructors.
